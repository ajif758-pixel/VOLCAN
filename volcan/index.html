Erupsi Nyata

Gunakan data historis aktivitas dan erupsi gunung api di Indonesia untuk mempelajari karakteristik letusan dan dampaknya pada wilayah sekitar.

  • Filter waktu kejadian
  • Pilih gunung api dari database
  • Pilih wilayah terdampak

CUSTOM
🎨

Skenario Hipotetik

Rancang simulasi letusan gunung api secara custom untuk menguji potensi bahaya, dampak, kesiapsiagaan, dan strategi mitigasi pada lokasi tertentu.

  • Lokasi target custom
  • Pilih gunung api
  • Atur intensitas erupsi (VEI)
  • Tentukan tinggi kolom erupsi
  • Arah angin dan jenis erupsi

Mulai Skenario Hipotetik

Klik tombol di bawah ini untuk membuat skenario erupsi custom. Anda dapat mengatur intensitas erupsi, tinggi kolom erupsi, arah angin, lokasi target, dan kondisi lingkungan secara bebas.
